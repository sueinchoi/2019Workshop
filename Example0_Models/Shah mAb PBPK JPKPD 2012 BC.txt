********** MODEL NAME

Shah et al mAb PBPK model 2012

********** MODEL NOTES

J Pharmacokinet Pharmacodyn (2012) 39:67-86 DOI 10.1007/s10928-011-9232-2

Towards a platform PBPK model to characterize the plasma and tissue
disposition of monoclonal antibodies in preclinical species and human.
Authors: Dhaval K. Shah, Alison M. Betts

Modifications of the model:
- Allow for bodyweight parameter and use allometric scaling with defined
  exponents to scale volumes and rates
- Including a tumor tissue
    - Constant tumor size
    - No drug binding to surface bound targets considered yet
    - Tumor volume and blood flow implemented with weight scaling
	  to be more consistent with other tissues and allow easier coding
- Dosing in mg and outputs for fitting in ug/mL
- Calculation of parameters based on function of others instead of
  numerical definition
- Coversion from ODE based format to biochemical notation (A+B <=> C)
  to allow easier model building and adding to the model

Input 1: IV
Input 2: SC

Notation:
- Species/States A_* in amount (mol) units and
- Concentrations calculated as variables C_* in concentration (M) units
- All reaction kinetics in amount/time (mol/h) unit

LU_CV       = LU_TV - LU_VV - LU_EV - LU_IV - LU_BV                   # (L) Lung Cellular volume
HE_CV       = HE_TV - HE_VV - HE_EV - HE_IV - HE_BV                   # (L) Heart Cellular volume
KI_CV       = KI_TV - KI_VV - KI_EV - KI_IV - KI_BV                   # (L) Kidney Cellular volume

SK_CV       = SK_TV - SK_VV - SK_EV - SK_IV - SK_BV                   # (L) Skin Cellular volume
BR_CV       = BR_TV - BR_VV - BR_EV - BR_IV - BR_BV                   # (L) Brain Cellular volume
AD_CV       = AD_TV - AD_VV - AD_EV - AD_IV - AD_BV                   # (L) Adipose Cellular volume
TH_CV       = TH_TV - TH_VV - TH_EV - TH_IV - TH_BV                   # (L) Thymus Cellular volume

LI_CV       = LI_TV - LI_VV - LI_EV - LI_IV - LI_BV                   # (L) Large intestine Cellular volume
SP_CV       = SP_TV - SP_VV - SP_EV - SP_IV - SP_BV                   # (L) Spleen Cellular volume
PA_CV       = PA_TV - PA_VV - PA_EV - PA_IV - PA_BV                   # (L) Pancreas Cellular volume
LIV_CV      = LIV_TV - LIV_VV - LIV_EV - LIV_IV - LIV_BV              # (L) Liver Cellular volume
BO_CV       = BO_TV - BO_VV - BO_EV - BO_IV - BO_BV                   # (L) Bone Cellular volume
OT_CV       = OT_TV - OT_VV - OT_EV - OT_IV - OT_BV                   # (L) Other Cellular volume
TUMOR_CV     = TUMOR_TV - TUMOR_VV - TUMOR_EV - TUMOR_IV - TUMOR_BV   # (L) TUMOR Cellular volume

A_LIV_V => A_LIV_I : vA_LIV_V_I
  vf = (1-LIV_VRC)*LIV_LF*C_LIV_V

A_LIV_V => A_LIV_E_U : vA_LIV_V_E_U
  vf = LIV_CL_UP*C_LIV_V


# Interstitial (same for all tissues)
# -----------------------------------

A_LIV_I => A_LIV_E_U : vA_LIV_I_E_U
  vf = LIV_CL_UP*C_LIV_I

A_LIV_I => A_LN : vA_LIV_I_LN
  vf = (1-LIV_ISRC)*LIV_LF*C_LIV_I


# Endosomal (same for all tissues)
# --------------------------------

A_LIV_E_B => A_LIV_V + A_LIV_FcRn : vA_LIV_E_B_V
  vf = LIV_CL_UP*FR*C_LIV_E_B

A_LIV_E_B => A_LIV_I + A_LIV_FcRn: vA_LIV_E_B_I
  vf = LIV_CL_UP*(1-FR)*C_LIV_E_B

A_LIV_E_U => : vA_LIV_E_U_deg
  vf = Kdeg_IgG*C_LIV_E_U*LIV_EV

A_LIV_E_U + A_LIV_FcRn <=> A_LIV_E_B : vA_LIV_E_U_E_B
  vf = (Kon_FcRn*C_LIV_E_U*(C_LIV_FcRn+FcRn))*LIV_EV
  vr = (Koff_FcRn*C_LIV_E_B)*LIV_EV
